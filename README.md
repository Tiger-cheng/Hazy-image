# WaterHaze Mini Benchmark

This repository provides a representative subset of the **WaterHaze** dataset proposed in our paper:

**MSAAFU-Net: A Multi-Scale Adaptive Attention U-Net for Water-Surface Image Dehazing**

## Overview

The WaterHaze dataset is designed for image dehazing in inland water-surface environments, including lakes, rivers, and artificial ponds. It contains paired hazy and haze-free images with representative floating debris such as:

- Plastic bottles
- Leaves
- Foam
- Domestic waste

The currently released subset is intended to facilitate reproducibility and fair comparison of dehazing algorithms.

## Current Release

The current version contains a representative subset of the complete dataset.

More image pairs and supplementary annotations will be uploaded gradually.

## Future Updates
The current repository contains only a representative subset. The complete WaterHaze dataset will be uploaded progressively after the associated paper is officially published.

- Additional paired hazy/clear image pairs
- More diverse water-surface scenes
- Richer environmental conditions
- Optional environmental annotations (e.g., haze density, reflection intensity, water quality)
